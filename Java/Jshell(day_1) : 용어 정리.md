## 매개변수 (parameter)
: 괄호안에 든 것이 무엇이든 매개변수(파라미터)라고 부름
<br/>
<br/> &#160; **예시**
<br/> &#160; ``` 5 * 5 = 25 ```
- 식 (expresions) : 5 * 5 = 25
- 비연산자 (operants) : 5
- 연산자 (operator) : * 

## 연산자 (operator) 
- 정수 (integers / whole numbers)
- 음수 (negative numbers)
- 곱셈 (Multiplication)0
- 덧셈 (addition)
- 뺄셈 (subtraction)
- 나눗셈 (division)
- 모듈러스 / 나머지값 (modulus)
- 비교연산자 (comparison operator) : ==
- 지정연산자(assignment operator : =
